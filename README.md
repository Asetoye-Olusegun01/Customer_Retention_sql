# 🧠 Customer Retention SQL Project

## 📋 Project Objective  
Analyze customer behavior to determine *retention* and *churn rates*, identify regions with high churn, and provide insights for improving customer loyalty and marketing strategies.

---

## 🗂 Dataset Overview  
| Column | Description |
|--------|--------------|
| customer_id | Unique customer identifier |
| signup_date | Date the customer joined |
| last_purchase_date | Date of the last purchase |
| region | Customer location |
| total_purchases | Number of purchases made |
| total_spent | Total money spent by the customer |

---

## 🧮 Key SQL Analysis  
- Classified customers as *Active* or *Churned* (based on 90-day activity threshold)  
- Calculated overall *Retention Rate* and *Churn Rate*  
- Compared *Churn by Region*  
- Analyzed *Average Spending* by customer status  
- Estimated *Customer Lifetime Value (CLV)*  

---

## 💡 Key Insights  
- *Retention Rate:* ~70%  
- *Churn Rate:* ~30%  
- *West Region* has the strongest customer retention  
- *Churned Customers* spend less overall  
- Focusing on re-engagement in high-churn regions can boost profits
